# Go Business Referral Dashboard
## Overview
A React-based Referral Dashboard application that allows users to:
* Login using API authentication
* View referral metrics and dashboard statistics
* View service summary information
* Copy referral link and referral code
* Search, sort, and paginate referrals
* View referral details
* Access protected routes using JWT authentication
## Technologies Used
* React JS
* React Router DOM
* JavaScript (ES6)
* CSS
* js-cookie
* React Icons
## Features
* Authentication with JWT Token
* Protected Routes
* Dashboard Overview
* Referral Management
* Search and Sorting
* Pagination
* Responsive UI
## Installation
1. Clone the repository
git clone <repository-url>
2. Install dependencies
npm install
3. Start the application
npm start
